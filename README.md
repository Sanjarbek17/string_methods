## String01

  A str of several words is given. All letters are lowercase. Return all letters to uppercase.

**Example 1:**

```Python
Input: a="google is a search engine"
Output: "GOOGLE IS A SEARCH ENGINE"

```

**Constraints:**

  - 2 <= length(a) <= 10^5
